## 🎯 Реализованные задачи (по ТЗ)

### ✅ Задача 1: Научные источники

Реализованы **6 источников**:

1. **CrossRef** — метаданные, DOI, аннотации, фильтрация по релевантности
2. **CyberLeninka** — российские научные статьи, PDF, полные тексты
3. **OpenAlex** — открытая база научных публикаций (162k+ результатов)
4. **PubMed** — медицинские и биологические статьи (NCBI)
5. **arXiv** — препринты по физике, математике, CS
6. **CiNii** — японские диссертации

**Результаты тестирования:**
- Запрос: `"thesis", "climate change"` (2020-2026, ru/en)
- Собрано: **18 документов** из 6 источников
- Средняя оценка качества: **0.96/1.0**
- Время выполнения: ~30-60 секунд

### ✅ Задача 2: Поиск в интернете (SERP)

- **DuckDuckGo HTML** — поиск веб-страниц без API-ключа
- Извлечение snippet, заголовков, URL
- Локализация по языку

**Результаты:** до 10-50 результатов с snippet и ссылками.

### ✅ Задача 3: Парсинг конкретных URL

- **Универсальный парсер** — извлечение текста из любых HTML-страниц
- Поддержка PDF-файлов (автоматическое извлечение текста)
- Извлечение заголовков, метаданных, основного контента

**Результаты:** полный текст страницы, разбитый на чанки для RAG.

## 📝 Формат выходных данных

Каждый документ содержит:
- `id` — уникальный идентификатор
- `title`, `url`, `authors`, `date`, `doi`
- `abstract` — аннотация
- `full_text_chunks[]` — чанки полного текста
- `language` — определённый язык
- `source` — источник (CrossRef, PubMed и т.д.)
- `files[]` — ссылки на PDF/файлы
- `metadata` — дополнительные метаданные
- `processing_info` — информация об обработке (validation_score, chunk_info)
